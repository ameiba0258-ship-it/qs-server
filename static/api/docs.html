🔌 API 接口文档

通过 API 接口查询企业/商家信息，支持高德/百度/腾讯地图数据

认证方式

所有 API 请求需要在 Header 中传递 API Key：

Authorization: Bearer YOUR_API_KEY

或者通过 URL 参数传递：

?api_key=YOUR_API_KEY
💡 API Key 可在管理后台生成，免费版每日 100 次，会员版不限

搜索接口

POST /api/search

请求参数

参数类型必填说明
keywordstring是搜索关键词（商家名称/品类）
providerstring否数据源: amap/baidumap/tencentmap (默认 amap)
provincestring否省份（如"北京市"、"广东省"）
citystring否城市（如"广州市"）
pageint否页码 (默认 1)
page_sizeint否每页条数 (默认 20, 最大 25)

请求示例

curl -X POST https://your-domain.com/api/search \\
  -H "Content-Type: application/json" \\
  -H "Authorization: Bearer YOUR_API_KEY" \\
  -d '{
    "keyword": "律师事务所",
    "province": "北京市",
    "provider": "amap",
    "page_size": 5
  }'

响应示例

{
  "success": true,
  "total": 600,
  "page": 1,
  "page_size": 5,
  "data": [
    {
      "company_name": "北京大成律师事务所",
      "phone": "010-58137799,13800138000",
      "mobile_phone": "13800138000",
      "landline_phone": "010-58137799",
      "address": "北京市朝阳区...",
      "industry": "商务服务业",
      "province": "北京市",
      "city": "北京市",
      "district": "朝阳区"
    }
  ]
}

批量搜索接口

POST /api/batch-search

参数同搜索接口，返回 task_id 用于轮询进度。

curl -X POST https://your-domain.com/api/batch-search \\
  -H "Content-Type: application/json" \\
  -H "Authorization: Bearer YOUR_API_KEY" \\
  -d '{"keyword": "律师事务所", "province": "广东省"}'

批量搜索进度

GET /api/batch-progress/{task_id} 
curl https://your-domain.com/api/batch-progress/abc123 \\
  -H "Authorization: Bearer YOUR_API_KEY"
